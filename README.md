## 🧭 Overview

agent-harness is a Windows app that helps you run a LangChain-based multi-agent system from one place. It uses agent routing, skill routing, and an execution harness to help tasks move to the right agent and the right tool.

This project is built for end users who want a simple way to launch and use an agent workflow without setting up each part by hand.

## 🧩 How It Works

agent-harness uses three main parts:

### Agent routing
The app sends each task to the agent best suited for it.

### Skill routing
The app checks which skill fits the task best.

### Execution harness
The app runs the task flow and keeps the work moving in the right order.

This setup helps keep work organized when the app handles more than one task or one agent at a time.
